# Harness Anything

> 将你的 AI 代理产生的**决策(decision)、任务(task)和事实(fact)**转化为 git 上的一等公民结构——可审查、可回滚、可重用——而不是让它们埋没在工作完成后就消失的聊天记录里。

当一个代理完成任务时，那些*推理过程*去哪里了？它为什么选择方案 A 而不是 B、尝试了什么、学到了什么——通常这些都存在没人再看一遍的记录文稿里。Harness Anything 将这些捕捉为持久的记录，你可以查询、挑战和基于它们继续工作。

三个原语，全部在你的仓库里版本化：

- **decision** ——为什么。一个选择、它的替代方案和证据。可回滚。
- **task** ——做什么。一个单位工作跨越六态生命周期。
- **fact** ——是什么。一个只增不改的观察，锚定到产生它的任务。

`ha` CLI 是你今天用的工具。它把纯 Markdown 写入你的 git 仓库，并维护可重建的 SQLite 投影以快速查询。
